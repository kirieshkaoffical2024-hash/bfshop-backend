┌────────────────────────────────────────────────────────────────────────────┐
│                    1️⃣ РЕГИСТРАЦИЯ                                         │
└────────────────────────────────────────────────────────────────────────────┘

1. Откройте: http://localhost:8080

2. Нажмите кнопку "Sign Up" (справа вверху)

3. Заполните форму:
   
   Логин:    testuser      (минимум 3 символа)
   Пароль:   test123456    (минимум 6 символов)
   Email:    (необязательно)

4. Нажмите "Sign Up"

5. Если успешно - вы автоматически войдёте в систему

✅ Теперь вы видите своё имя справа вверху!


┌────────────────────────────────────────────────────────────────────────────┐
│                    2️⃣ СОЗДАНИЕ ТОВАРА                                     │
└────────────────────────────────────────────────────────────────────────────┘

1. Нажмите "Sell" в меню (вверху)

2. Выберите тип товара:
   
   🎮 Account      - Продажа аккаунта
   ⚡ Boost        - Прокачка уровня
   🗡️ Quest        - Прохождение квестов/рейдов
   👹 Boss         - Призыв боссов

3. Заполните форму:
   
   Title:        Название товара
   Description:  Подробное описание
   Payment Fruit: Какой фрукт хотите получить
   Amount:       Сколько фруктов

4. Заполните дополнительные поля (зависят от типа)

5. Нажмите "Create Listing"

✅ Товар создан! Вы увидите его на главной странице.


┌────────────────────────────────────────────────────────────────────────────┐
│                    3️⃣ ПОКУПКА ТОВАРА                                      │
└────────────────────────────────────────────────────────────────────────────┘

1. Перейдите на главную страницу (нажмите "Home")

2. Найдите товар, который хотите купить

3. Кликните на карточку товара

4. Нажмите "Buy Now"

5. Подтвердите покупку

✅ Откроется чат с продавцом!


┌────────────────────────────────────────────────────────────────────────────┐
│                    4️⃣ ЧАТ С ПРОДАВЦОМ                                     │
└────────────────────────────────────────────────────────────────────────────┘

После покупки откроется чат:

1. Обсудите с продавцом детали оплаты
   
   Например:
   "Привет! Я готов отправить 5x Leopard. Куда отправить?"

2. Продавец ответит:
   
   "Отправь на мой аккаунт: username123"

3. Отправьте фрукт В ИГРЕ (не на сайте!)

4. Напишите в чате:
   
   "Отправил 5x Leopard!"

5. Продавец проверит и нажмёт "Confirm Order"

✅ Заказ завершён!


┌────────────────────────────────────────────────────────────────────────────┐
│                    5️⃣ ПОДТВЕРЖДЕНИЕ ЗАКАЗА (ДЛЯ ПРОДАВЦА)                │
└────────────────────────────────────────────────────────────────────────────┘

Если вы продавец:

1. Покупатель создаст заказ

2. Вы получите уведомление (откроется чат)

3. Обсудите детали оплаты

4. Получите фрукт В ИГРЕ

5. Нажмите "Confirm Order" в чате

✅ Заказ завершён! Покупатель может оставить отзыв.


┌────────────────────────────────────────────────────────────────────────────┐
│                    6️⃣ ОТЗЫВЫ                                              │
└────────────────────────────────────────────────────────────────────────────┘

После завершения заказа:

1. Покупатель видит форму отзыва в чате

2. Выберите количество звёзд (1-5):
   
   ☆☆☆☆☆  → Кликните на звёзды

3. Напишите комментарий (необязательно)

4. Нажмите "Submit Review"

✅ Отзыв появится на профиле продавца!


┌────────────────────────────────────────────────────────────────────────────┐
│                    7️⃣ ПРОСМОТР ПРОФИЛЯ                                    │
└────────────────────────────────────────────────────────────────────────────┘

1. Нажмите на своё имя (справа вверху)

2. Вы увидите:
   
   - Ваш аватар
   - Рейтинг (звёзды)
   - Количество товаров
   - Количество продаж
   - Отзывы

3. Можете загрузить аватар:
   
   Нажмите "Change Avatar" → Выберите файл

✅ Профиль обновлён!


┌────────────────────────────────────────────────────────────────────────────┐
│                    8️⃣ МОИ ЗАКАЗЫ                                          │
└────────────────────────────────────────────────────────────────────────────┘

1. Нажмите "My Orders" в меню

2. Две вкладки:
   
   Buying:  Заказы, где вы покупатель
   Selling: Заказы, где вы продавец

3. Для каждого заказа можете:
   
   - Посмотреть детали
   - Открыть чат
   - Оставить отзыв (если завершён)

✅ Вся история заказов в одном месте!


┌────────────────────────────────────────────────────────────────────────────┐
│                    🎯 ПОЛНЫЙ ЦИКЛ СДЕЛКИ                                   │
└────────────────────────────────────────────────────────────────────────────┘

ПРОДАВЕЦ:
1. Создаёт товар (Sell)
2. Ждёт покупателя
3. Получает уведомление о покупке
4. Обсуждает детали в чате
5. Получает фрукт в игре
6. Подтверждает заказ (Confirm Order)
7. Получает отзыв

ПОКУПАТЕЛЬ:
1. Находит товар на главной
2. Покупает (Buy Now)
3. Обсуждает детали в чате
4. Отправляет фрукт в игре
5. Ждёт подтверждения
6. Получает товар
7. Оставляет отзыв


┌────────────────────────────────────────────────────────────────────────────┐
│                    💡 СОВЕТЫ                                               │
└────────────────────────────────────────────────────────────────────────────┘

✅ Всегда обсуждайте детали в чате перед отправкой фруктов

✅ Не отправляйте фрукты до обсуждения с продавцом

✅ Продавец должен подтвердить получение фруктов

✅ Оставляйте честные отзывы - это помогает другим

✅ Проверяйте рейтинг продавца перед покупкой


┌────────────────────────────────────────────────────────────────────────────┐
│                    🔍 ФИЛЬТРЫ И ПОИСК                                      │
└────────────────────────────────────────────────────────────────────────────┘

На главной странице:

1. Фильтры по типу:
   
   All       - Все товары
   Accounts  - Только аккаунты
   Boosts    - Только прокачка
   Quests    - Только квесты
   Bosses    - Только боссы

2. Поиск:
   
   Введите название в строку поиска
   Нажмите "Search"

✅ Быстро находите нужные товары!
