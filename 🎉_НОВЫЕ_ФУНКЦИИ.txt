🎮 ПРОДАЖА АККАУНТА:

1. Нажмите "Продать" → "Создать объявление"
2. Выберите тип: "Аккаунт"
3. Заполните информацию:
   - Название: "Макс уровень 2550 с Тигром"
   - Описание: "Полностью прокачанный аккаунт..."
   - Уровень: 2550
   - Фрукты: Тигр, Дракон, Тесто
   - Бели: 1000000000
   - **ДАННЫЕ АККАУНТА:**
     ```
     Логин: myaccount123
     Пароль: SecurePass456
     Email: example@mail.com
     ```
4. Выберите фрукт для оплаты: Дракон
5. Количество: 2
6. Создать объявление

⚠️ Данные аккаунта будут показаны покупателю ТОЛЬКО после того,
   как вы подтвердите получение фрукта!

💰 ПРОЦЕСС ПОКУПКИ:

1. Покупатель нажимает "Купить сейчас"
2. Открывается чат с продавцом
3. Покупатель договаривается о передаче фрукта
4. Покупатель отправляет фрукт в игре
5. Продавец получает фрукт и нажимает "Подтвердить заказ"
6. Данные аккаунта автоматически появляются в чате!
7. Покупатель может оставить отзыв со звездами

⭐ СИСТЕМА ОТЗЫВОВ:

1. После завершения сделки появляется форма отзыва
2. Выберите количество звезд (1-5)
3. Напишите комментарий
4. Нажмите "Отправить отзыв"
5. Отзыв появится в профиле продавца
6. Рейтинг продавца обновится автоматически
