# 🐘 Установка PostgreSQL на Windows

## 📥 Способ 1: Официальный установщик (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Скачать PostgreSQL

1. Откройте браузер и перейдите на:
   ```
   https://www.postgresql.org/download/windows/
   ```

2. Нажмите **"Download the installer"**

3. Выберите последнюю версию (например, PostgreSQL 16.x)

4. Скачайте установщик для Windows x86-64

### Шаг 2: Установить PostgreSQL

1. **Запустите скачанный файл** (postgresql-16.x-windows-x64.exe)

2. **Нажмите "Next"** на экране приветствия

3. **Выберите папку установки** (оставьте по умолчанию)
   ```
   C:\Program Files\PostgreSQL\16
   ```

4. **Выберите компоненты** (оставьте все галочки):
   - ✅ PostgreSQL Server
   - ✅ pgAdmin 4
   - ✅ Stack Builder
   - ✅ Command Line Tools

5. **Выберите папку для данных** (оставьте по умолчанию)
   ```
   C:\Program Files\PostgreSQL\16\data
   ```

6. **ВАЖНО! Установите пароль для пользователя postgres**
   ```
   Введите пароль: ваш_пароль
   Подтвердите: ваш_пароль
   
   ⚠️ ЗАПОМНИТЕ ЭТОТ ПАРОЛЬ! Он понадобится для .env файла
   ```

7. **Выберите порт** (оставьте 5432)

8. **Выберите локаль** (оставьте по умолчанию)

9. **Нажмите "Next"** и дождитесь установки

10. **Снимите галочку** "Launch Stack Builder" и нажмите "Finish"

### Шаг 3: Проверить установку

Откройте командную строку (cmd) и выполните:

```bash
psql --version
```

Должно показать версию PostgreSQL.

### Шаг 4: Создать базу данных

1. Откройте **SQL Shell (psql)** из меню Пуск

2. Нажимайте Enter на всех вопросах (используются значения по умолчанию):
   ```
   Server [localhost]:
   Database [postgres]:
   Port [5432]:
   Username [postgres]:
   ```

3. Введите **пароль**, который вы установили при установке

4. Создайте базу данных:
   ```sql
   CREATE DATABASE bfshop;
   ```

5. Проверьте, что база создана:
   ```sql
   \l
   ```
   (должна появиться bfshop в списке)

6. Выйдите:
   ```sql
   \q
   ```

### Шаг 5: Применить схему базы данных

Откройте командную строку (cmd) в папке BFshop:

```bash
cd C:\Users\USER-PC\Desktop\sssssss\BFshop

psql -U postgres -d bfshop < database\schema.sql
```

Введите пароль postgres, когда попросит.

### Шаг 6: Настроить .env файл

1. Откройте файл `backend\.env` (или создайте из .env.example)

2. Измените строку DATABASE_URL:
   ```env
   DATABASE_URL=postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/bfshop
   ```
   
   Замените `ВАШ_ПАРОЛЬ` на пароль, который вы установили

3. Сохраните файл

### Шаг 7: Готово! Запустите BFshop

```bash
START_BFSHOP.bat
```

## 🐛 Решение проблем

### Проблема: "psql не является внутренней командой"

**Решение:** Добавьте PostgreSQL в PATH:

1. Откройте **Панель управления** → **Система** → **Дополнительные параметры системы**
2. Нажмите **"Переменные среды"**
3. В разделе **"Системные переменные"** найдите **Path**
4. Нажмите **"Изменить"**
5. Нажмите **"Создать"** и добавьте:
   ```
   C:\Program Files\PostgreSQL\16\bin
   ```
6. Нажмите **OK** везде
7. **Перезапустите командную строку**

### Проблема: "Не могу подключиться к PostgreSQL"

**Решение 1:** Проверьте, что служба запущена:

1. Откройте **Диспетчер задач** (Ctrl+Shift+Esc)
2. Вкладка **"Службы"**
3. Найдите **postgresql-x64-16**
4. Если остановлена, нажмите правой кнопкой → **"Запустить"**

**Решение 2:** Запустите службу через командную строку (от администратора):

```bash
net start postgresql-x64-16
```

### Проблема: "Забыл пароль postgres"

**Решение:** Сбросьте пароль:

1. Найдите файл `pg_hba.conf`:
   ```
   C:\Program Files\PostgreSQL\16\data\pg_hba.conf
   ```

2. Откройте в блокноте (от администратора)

3. Найдите строки с `md5` и замените на `trust`:
   ```
   # Было:
   host    all             all             127.0.0.1/32            md5
   
   # Стало:
   host    all             all             127.0.0.1/32            trust
   ```

4. Сохраните файл

5. Перезапустите PostgreSQL:
   ```bash
   net stop postgresql-x64-16
   net start postgresql-x64-16
   ```

6. Откройте psql (пароль не нужен):
   ```bash
   psql -U postgres
   ```

7. Измените пароль:
   ```sql
   ALTER USER postgres PASSWORD 'новый_пароль';
   ```

8. Верните `md5` обратно в `pg_hba.conf`

9. Перезапустите PostgreSQL снова

## 🎯 Быстрая проверка

После установки выполните:

```bash
# 1. Проверка версии
psql --version

# 2. Проверка подключения
psql -U postgres -c "SELECT version();"

# 3. Проверка базы данных
psql -U postgres -c "\l" | findstr bfshop
```

Если всё работает, вы увидите версию PostgreSQL и список баз данных.

## 📚 Полезные команды

```bash
# Запустить службу PostgreSQL
net start postgresql-x64-16

# Остановить службу
net stop postgresql-x64-16

# Подключиться к базе данных
psql -U postgres -d bfshop

# Список баз данных
psql -U postgres -c "\l"

# Список таблиц в bfshop
psql -U postgres -d bfshop -c "\dt"

# Удалить базу данных (если нужно пересоздать)
psql -U postgres -c "DROP DATABASE bfshop;"
psql -U postgres -c "CREATE DATABASE bfshop;"
```
