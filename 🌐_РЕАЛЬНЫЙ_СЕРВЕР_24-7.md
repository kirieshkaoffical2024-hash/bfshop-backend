## ⚡ БЫСТРЫЙ СПОСОБ: Railway (Рекомендуется!)

### Почему Railway?
- ✅ **Бесплатно** (500 часов в месяц)
- ✅ **Очень просто** (5 минут настройки)
- ✅ **Автоматический деплой** (загрузил код - всё работает)
- ✅ **PostgreSQL включен**
- ✅ **HTTPS автоматически**
- ✅ **Работает 24/7**

### Шаг 1: Регистрация

1. Зайдите на https://railway.app
2. Нажмите "Start a New Project"
3. Войдите через GitHub (создайте аккаунт если нет)

### Шаг 2: Создание проекта

1. Нажмите "New Project"
2. Выберите "Deploy PostgreSQL"
3. Подождите 1 минуту пока создастся база данных

### Шаг 3: Получение DATABASE_URL

1. Кликните на PostgreSQL в вашем проекте
2. Перейдите на вкладку "Connect"
3. Скопируйте "Postgres Connection URL"
4. Сохраните его - понадобится позже

### Шаг 4: Загрузка Backend

#### 4.1 Создайте GitHub репозиторий

```bash
cd BFshop/backend
git init
git add .
git commit -m "Initial commit"
```

Создайте репозиторий на GitHub:
1. Зайдите на https://github.com
2. Нажмите "New repository"
3. Назовите: `bfshop-backend`
4. Нажмите "Create repository"

Загрузите код:
```bash
git remote add origin https://github.com/ВАШ_НИК/bfshop-backend.git
git branch -M main
git push -u origin main
```

#### 4.2 Деплой на Railway

1. В Railway нажмите "New" → "GitHub Repo"
2. Выберите `bfshop-backend`
3. Railway автоматически определит Node.js проект

#### 4.3 Настройка переменных окружения

В Railway перейдите в настройки проекта → Variables:

```
DATABASE_URL = (вставьте URL из шага 3)
JWT_SECRET = ваш-секретный-ключ-12345
NODE_ENV = production
PORT = 3000
```

#### 4.4 Применение миграций базы данных

В Railway откройте терминал и выполните:

```bash
# Подключитесь к PostgreSQL
psql $DATABASE_URL

# Создайте таблицы
\i database/schema.sql
\i database/add_support_system.sql
\i database/update_schema.sql

# Выйдите
\q
```

Или загрузите SQL файлы через Railway Dashboard.

### Шаг 5: Загрузка Frontend

#### 5.1 Обновите API_URL

Откройте `BFshop/frontend/assets/js/app.js`:

```javascript
// Замените localhost на ваш Railway URL
const API_URL = 'https://ваш-backend.railway.app/api';
```

Узнать URL backend:
1. Откройте ваш backend проект в Railway
2. Перейдите в Settings → Domains
3. Скопируйте URL (например: `bfshop-backend-production.up.railway.app`)

#### 5.2 Создайте GitHub репозиторий для frontend

```bash
cd BFshop/frontend
git init
git add .
git commit -m "Initial commit"
```

Создайте репозиторий `bfshop-frontend` на GitHub и загрузите:

```bash
git remote add origin https://github.com/ВАШ_НИК/bfshop-frontend.git
git branch -M main
git push -u origin main
```

#### 5.3 Деплой frontend на Railway

1. В Railway нажмите "New" → "GitHub Repo"
2. Выберите `bfshop-frontend`
3. Railway определит статический сайт

### Шаг 6: Получите ссылку!

1. Откройте frontend проект в Railway
2. Перейдите в Settings → Domains
3. Скопируйте URL: `https://ваш-frontend.railway.app`

**🎉 Готово! Отправьте эту ссылку друзьям!**

## 🔥 АЛЬТЕРНАТИВА: Render.com

### Почему Render?
- ✅ Бесплатно навсегда
- ✅ Простая настройка
- ✅ PostgreSQL включен
- ⚠️ Засыпает после 15 минут неактивности (первый запрос медленный)

### Шаг 1: Регистрация

1. Зайдите на https://render.com
2. Нажмите "Get Started"
3. Войдите через GitHub

### Шаг 2: Создание PostgreSQL

1. Нажмите "New +" → "PostgreSQL"
2. Имя: `bfshop-db`
3. Plan: Free
4. Нажмите "Create Database"
5. Скопируйте "Internal Database URL"

### Шаг 3: Деплой Backend

1. Нажмите "New +" → "Web Service"
2. Подключите GitHub репозиторий `bfshop-backend`
3. Настройки:
   - Name: `bfshop-backend`
   - Environment: Node
   - Build Command: `npm install`
   - Start Command: `node server.js`
   - Plan: Free

4. Environment Variables:
```
DATABASE_URL = (Internal Database URL)
JWT_SECRET = ваш-секретный-ключ
NODE_ENV = production
```

5. Нажмите "Create Web Service"

### Шаг 4: Деплой Frontend

1. Обновите `API_URL` в `app.js` на URL вашего backend
2. Нажмите "New +" → "Static Site"
3. Подключите `bfshop-frontend`
4. Build Command: (оставьте пустым)
5. Publish Directory: `.`
6. Нажмите "Create Static Site"

### Шаг 5: Готово!

Ваш сайт доступен по ссылке из Render Dashboard!

## 💎 ПРЕМИУМ: VPS (Полный контроль)

### Если нужен полный контроль и высокая производительность:

#### Провайдеры:
- **DigitalOcean** - $5/месяц
- **Vultr** - $5/месяц  
- **Linode** - $5/месяц
- **Hetzner** - €4/месяц (дешевле всех!)

### Быстрая настройка на Ubuntu:

```bash
# 1. Подключитесь к серверу
ssh root@ваш_ip

# 2. Установите Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 3. Установите PostgreSQL
sudo apt install postgresql postgresql-contrib

# 4. Настройте PostgreSQL
sudo -u postgres psql
CREATE DATABASE bfshop;
CREATE USER bfshop WITH PASSWORD 'ваш_пароль';
GRANT ALL PRIVILEGES ON DATABASE bfshop TO bfshop;
\q

# 5. Клонируйте проект
git clone https://github.com/ВАШ_НИК/bfshop-backend.git
cd bfshop-backend
npm install

# 6. Настройте .env
nano .env
# Вставьте:
DATABASE_URL=postgresql://bfshop:ваш_пароль@localhost:5432/bfshop
JWT_SECRET=ваш-секретный-ключ
PORT=3000

# 7. Примените миграции
psql -U bfshop -d bfshop -f database/schema.sql
psql -U bfshop -d bfshop -f database/add_support_system.sql

# 8. Установите PM2 (для работы 24/7)
sudo npm install -g pm2
pm2 start server.js --name bfshop-backend
pm2 startup
pm2 save

# 9. Установите Nginx
sudo apt install nginx

# 10. Настройте Nginx
sudo nano /etc/nginx/sites-available/bfshop

# Вставьте:
server {
    listen 80;
    server_name ваш_домен.com;

    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }

    location / {
        root /var/www/bfshop-frontend;
        try_files $uri $uri/ /index.html;
    }
}

# Активируйте конфиг
sudo ln -s /etc/nginx/sites-available/bfshop /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx

# 11. Загрузите frontend
cd /var/www
sudo git clone https://github.com/ВАШ_НИК/bfshop-frontend.git
sudo chown -R www-data:www-data bfshop-frontend

# 12. Установите SSL (HTTPS)
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d ваш_домен.com
```

**Готово! Сайт работает 24/7 на вашем сервере!**

## 📞 Поддержка

### Если что-то не работает:

1. **Проверьте логи** в Railway/Render Dashboard
2. **Проверьте переменные окружения**
3. **Убедитесь что DATABASE_URL правильный**
4. **Проверьте что таблицы созданы**

### Полезные команды для проверки:

```bash
# Проверить подключение к БД
psql $DATABASE_URL -c "SELECT version();"

# Проверить таблицы
psql $DATABASE_URL -c "\dt"

# Проверить пользователей
psql $DATABASE_URL -c "SELECT username, is_admin FROM users;"
```
