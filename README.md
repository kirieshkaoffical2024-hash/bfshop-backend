# 🎮 BFshop - Blox Fruits Marketplace

Полноценный маркетплейс для торговли аккаунтами и услугами Blox Fruits.

## 🚀 Особенности

✅ **Регистрация и авторизация** - логин/пароль, аватарка
✅ **Сессии** - аккаунт не выходит при перезагрузке
✅ **Реальная база данных** - PostgreSQL/MySQL
✅ **Торговля за фрукты** - без реальных денег!
✅ **Чат** - общение продавца и покупателя
✅ **Типы товаров**:
  - Аккаунты
  - Прокачка
  - Прохождение квестов (рейды, данжи)
  - Призыв боссов
✅ **Отзывы** - рейтинг 0-5 звезд
✅ **Современный дизайн** - темная тема, анимации

## 🛠️ Установка

1. Установите Node.js
2. Установите PostgreSQL
3. Запустите: `npm install`
4. Настройте `.env`
5. Запустите: `npm start`
