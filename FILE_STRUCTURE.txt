╔════════════════════════════════════════════════════════════════╗
║                   🚀 Quick Start Commands                      ║
╚════════════════════════════════════════════════════════════════╝

Windows (Automatic):
  > START_BFSHOP.bat

Manual:
  1. cd BFshop/backend
  2. npm install
  3. npm start
  4. (New terminal) cd ../frontend
  5. python -m http.server 8080
  6. Open: http://localhost:8080

Deploy to Railway:
  1. railway login
  2. railway init
  3. railway add postgresql
  4. railway up
