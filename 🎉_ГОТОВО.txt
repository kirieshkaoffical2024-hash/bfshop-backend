╔═══════════════════════════════════════════════════════════════════════════╗
║                    ВАРИАНТ 2: РУЧНОЙ ЗАПУСК                               ║
╚═══════════════════════════════════════════════════════════════════════════╝

1. Установите зависимости:
   cd BFshop/backend
   npm install

2. Настройте PostgreSQL:
   psql -U postgres
   CREATE DATABASE bfshop;
   \q
   psql -U postgres -d bfshop < ../database/schema.sql

3. Настройте .env:
   cd backend
   copy .env.example .env
   notepad .env
   
   Измените:
   DATABASE_URL=postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/bfshop
   JWT_SECRET=случайный-ключ
   SESSION_SECRET=случайный-ключ

4. Запустите backend:
   npm start

5. Запустите frontend (в новом окне):
   cd ../frontend
   python -m http.server 8080

6. Откройте браузер:
   http://localhost:8080

╔═══════════════════════════════════════════════════════════════════════════╗
║                    Railway (БЕСПЛАТНО!) ⭐                                 ║
╚═══════════════════════════════════════════════════════════════════════════╝

1. Зарегистрируйтесь: https://railway.app

2. Установите CLI:
   npm install -g @railway/cli

3. Войдите:
   railway login

4. Создайте проект:
   cd BFshop/backend
   railway init

5. Добавьте PostgreSQL:
   railway add postgresql

6. Деплой:
   railway up

7. Получите URL:
   railway domain

ГОТОВО! Ваш сайт доступен по ссылке!

1. РЕГИСТРАЦИЯ
   → Откройте http://localhost:8080/login.html
   → Нажмите "Sign Up"
   → Введите логин и пароль
   → Загрузите аватар (опционально)

2. СОЗДАНИЕ ТОВАРА
   → Нажмите "Sell" в меню
   → Выберите тип (Аккаунт/Прокачка/Квест/Босс)
   → Заполните описание
   → Укажите цену в фруктах
   → Отправьте

3. ПОКУПКА
   → Найдите товар на главной
   → Нажмите на товар
   → Нажмите "Buy Now"
   → Подтвердите покупку

4. ЧАТ
   → Обсудите оплату с продавцом
   → Покупатель отправляет фрукт В ИГРЕ
   → Продавец подтверждает получение

5. ПОДТВЕРЖДЕНИЕ
   → Продавец нажимает "Confirm Order"
   → Заказ завершается

6. ОТЗЫВ
   → Покупатель ставит оценку 0-5 звёзд
   → Пишет комментарий
   → Отправляет

❌ Ошибка подключения к БД:
   → Проверьте, что PostgreSQL запущен
   → Проверьте DATABASE_URL в .env

❌ Порт занят:
   → netstat -ano | findstr :3000
   → taskkill /PID <PID> /F

❌ CORS ошибка:
   → Проверьте FRONTEND_URL в .env

❌ WebSocket не подключается:
   → Проверьте SOCKET_URL в chat-client.js

❌ Не загружается аватар:
   → mkdir backend\uploads
